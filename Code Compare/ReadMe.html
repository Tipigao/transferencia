Overview

Code Compare is an efficient tool for folder synchronization and source comparison and merging, that takes into account specific of programming languages (supports C#, Visual Basic, XML, XAML, etc.) and offers two- and three- way comparison. Code Compare can be integrated to the Visual Studio and it uses primary features of Visual Studio text editor.


Features

Code Compare offers to you the following features:

  • Flawless Visual Studio integration
  • Advanced comparison and merging functionality
  • Easy source control system integration with the help of command line
  • Convenient user interface

Requirements

Code Compare requires the following:

  • .NET Framework 4.0, 4.5, or 4.6 installed.

    • You can download .NET Framework here

    Note, .NET Framework 4.5 is pre-installed in Windows 8.

  • 800 MHz or higher processor
  • 512 MB RAM
  • 30 MB of free hard disk space
  • Microsoft Windows XP/2003/2008/Vista/Windows 7 and Windows 8

Installation

Code Compare contains all required files and documentation. By default all files are installed in the "Program Files\Devart\CodeCompare" folder.


Documentation

Code Compare includes complete documentation on its features. You can access it from the "Tools -> Code Compare -> Help" menu item, or from the Start menu.
